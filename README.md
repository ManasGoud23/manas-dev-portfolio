## ✨ Features

- **🎨 Modern Glassmorphic Design:** Sleek dark mode aesthetics with neon glows, shimmer animations, and fluid typography.
- **📱 Fully Responsive:** Carefully optimized layouts for Desktop, Tablet, and Mobile devices (including a custom mobile hamburger menu).
- **🖱️ Interactive UI:** Custom cursor tracking, scroll-reveal IntersectionObserver animations, and dynamic hover states.
- **📊 Skills Dashboard:** Premium cards highlighting core competencies in Data Structures, AI Prompting, and Full-Stack Architecture, with animated progress bars.
- **📧 Direct Contact Form:** Integrated Web3Forms API for instant, backend-less email delivery. No email client pop-ups!
- **⚡ High Performance:** Pure HTML, CSS, and Vanilla JavaScript—no heavy frameworks attached, ensuring lightning-fast load times.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Styling:** CSS Variables, Flexbox, CSS Grid, Keyframe Animations
- **Integrations:** [Web3Forms](https://web3forms.com/) for contact form

## 🚀 Quick Start (Local Development)

Since this is a static website, you don't need any complex build tools. Just serve the directory!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ManasGoud23/manas-dev-portfolio.git
   ```
2. **Navigate to the project:**
   ```bash
   cd manas-dev-portfolio
   ```
3. **Run a local server** (using `npx http-server`, `Live Server` in VS Code, or Python):
   ```bash
   # Using npx (Node.js)
   npx http-server . -p 8080
   
   # Or using Python 3
   python -m http.server 8080
   ```
4. **Open in browser:** Visit `http://localhost:8080`

## 📫 Contact Setup

If you fork this project, be sure to update the **Web3Forms API key** to receive messages to your own email:
1. Go to [Web3Forms](https://web3forms.com/).
2. Enter your email and get your free Access Key.
3. Replace the `WEB3FORMS_KEY` variable in `index.html` (around line 2726) with your new key.
